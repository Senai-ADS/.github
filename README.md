# 🤝 Contributing

To contribute to the projects:

1. Fork the repository
2. Create a branch

```bash
git checkout -b feature-name
```

3. Commit your changes

```bash
git commit -m "Add new feature"
```

4. Push to GitHub

```bash
git push origin feature-name
```

5. Open a Pull Request
